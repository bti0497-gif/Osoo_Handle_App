# 도로공사 연동 이어서 진행 계획

## 1. 오늘의 종료 지점
- 로그인 자동주입 경로와 DOM 덤프 헬퍼는 기본 정리 완료
- 실제 도로공사 로그인 화면에서 자동입력 검증이 남아 있는 상태
- 문자 인증(SMS) 단계 포함 실제 수동 검증이 아직 미완료

## 2. 내일 바로 수행할 작업
1. 도로공사 실제 로그인 페이지를 열고 수동 로그인 검증 수행
2. SMS 인증까지 통과한 뒤 실제 화면 구조를 확인
3. DOM 덤프 파일을 저장하여 입력 필드/버튼/폼 구조를 캡처
4. 캡처 결과를 기준으로 필드 의미 기반 매핑 규칙을 설계
5. JSON 입력 방식과 DB 저장 경로를 함께 연결해 배포용 흐름으로 정리

## 3. 확인 포인트
- 로그인 화면의 입력 필드가 안정적으로 탐지되는가
- SMS 인증 후 화면 전환 후에도 구조가 정상적으로 캡처되는가
- 이후 자동 바인딩에 필요한 핵심 요소가 모두 식별되는가

## 4. 완료 기준
- 수동 로그인 + SMS 인증 성공
- 실제 화면 DOM 구조 수집 완료
- 다음 단계의 자동 바인딩 설계 기준이 문서화 완료
