# 소통게시판 팝업 공지 보호 계약

- 팝업 지정은 `admin`, `group_admin`, `super_admin`, `central_admin` 역할만 할 수 있으며 서버에서 권한을 다시 검사한다.
- 팝업 대상은 기존 게시판 `visible_sites`/`target_site` 가시성 결과를 그대로 사용한다. 전체 현장과 특정 현장 규칙을 별도로 우회하지 않는다.
- 팝업 게시 기간은 1일 이상 7일 이하이고 기본값은 1일이다. 서버가 클라이언트 요청값을 1~7 범위로 강제한다.
- 만료된 글은 삭제하지 않지만 조회 시 `is_popup=false`로 정규화하여 일반 게시글로 남긴다.
- 현장 사용자가 선택한 `다시 보지 않기`는 사용자·현장·게시글별 로컬 설정으로만 저장한다.
- 팝업 조회 실패는 로그인이나 대시보드 진입을 막지 않는다.
