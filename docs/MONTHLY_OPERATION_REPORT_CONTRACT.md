# 월운영보고서 보호 계약

- 기본양식은 `templates/reports/월운영보고서.xlsx`이며 기존 시트, 셀 서식, 병합, 인쇄영역과 수식을 수정하지 않는다.
- 데이터는 엑셀에 정의된 이름 범위에만 기록한다. 알 수 없는 시트와 향후 추가된 시트는 건드리지 않는다.
- 출력 파일에는 `fullCalcOnLoad`, `forceFullCalc`, `calcMode=auto`를 설정해 Excel을 열 때 기존 수식을 전체 재계산한다.
- 앱 데이터 폴더에 사용자가 수정한 `월운영보고서.xlsx`가 있으면 업데이트 또는 시작 동기화가 덮어쓰지 않는다. 기본양식은 파일이 없을 때만 복사한다.
- 양식의 필수 이름 범위가 빠졌거나 크기가 달라지면 잘못된 셀에 쓰지 않고 생성을 중단한다.
- 릴리스 검증 전에 `node scripts/validate-monthly-operation-report.cjs`를 실행하여 수식, 병합, 인쇄영역 및 계산 설정 보존을 확인한다.
