# QnTECH 현장 ID 관리

## 관리 원칙

- Google Sheets의 `KMSC`와 `Wastewater_Sites` 시트에 `qntech_site_id` 열을 둔다.
- QnTECH 로그인 계정이 같더라도 방향별 현장은 서로 다른 `qntech_site_id`로 구분한다.
- admin이 현장을 확정하면 해당 현장의 QnTECH 계정과 `qntech_site_id`를 로컬 DB에 함께 저장한다.
- 수질 데이터와 사진을 가져올 때는 로그인 후 반환된 첫 번째 현장을 사용하지 않는다.
- 로컬에 저장된 `qntech_site_id`와 일치하는 QnTECH 현장만 조회한다.
- ID가 없거나 로그인 계정이 해당 ID에 접근할 수 없으면 가져오기를 중단한다.
